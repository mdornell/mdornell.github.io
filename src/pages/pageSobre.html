Unidades

As unidades foram utilizadas em toda a página devido à sua utilização para dimensionar os itens da tela, sendo unidades absolutas ou relativas.

Unidades absolutas: foi utilizado o Pixel (px). Já as relativas foram utilizados dois tipos: Porcentagem (%) e Viewport Width e Viewport Height (vw e vh).

Mídia Query

A mídia query foi utilizada para criar a responsividade da página, permitindo que ela seja utilizada em diversos aparelhos.

O limite de largura que foi utilizado para determinar o ponto de quebra foi de 1000px e pode ser encontrado no arquivo _responsividade.scss.

Animiações

A utilização de animação foi feita para a imagem que está no primeiro conteúdo da página, aplicando um movimento vertical que oscila de cima para baixo e vice-versa.

Sendo encontrado no arquivo _topoSite.scss.

Tranformação

As transformações foram utilizadas em sua maioria para o crescimento do item ao passar do mouse utilizando a função scale. Sendo encontradas em diversas partes da página.

Podem ser encontradas no arquivo _especialidades.scss.

SASS

Toda a estilização da página foi feita com arquivos SASS, facilitando assim o desenvolvimento. Foram utilizadas variáveis, mixins e o encapsulamento de blocos, facilitando a organização do código.

Os arquivos estão na pasta styles, a qual está dividida entre a página principal e a página "sobre" (esta página).

Flex Box

Os parâmetros que envolvem flexbox foram utilizados em diversas partes, como justify-content, display flex e outros.

Pode ser visto no arquivo _cabecalho.scss logo no início do código.

Grid Layout

O Grid Layout foi utilizado na seção que descreve minhas curiosidades, trazendo um grid vertical.

Descrito no arquivo _curiosidades.scss.
